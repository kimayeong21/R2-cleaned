# R2 로보테크 AI 프로젝트 포트폴리오

## 1. 탑뷰 미로 탈출 게임

터미널 기반 탑뷰 시점의 미로 탈출 게임입니다. MVC 구조를 참고해 `Map`, `Monster`, `Player`, `Skill`, `Item` 모듈을 나누고, 아스키 아트와 아이템 시스템을 통해 텍스트 기반 게임의 표현력을 높였습니다.

### 담당 역할

- Player 클래스 및 View 파트 구현
- 키 입력 처리
- 플레이어 이동, 공격, 피격 처리
- 스테이지 종료 조건 판정
- 플레이어 렌더링 처리

### 주요 기능

- txt 파일 기반 20x20 맵 데이터 로드
- 몬스터 랜덤 이동
- 회복, 공격력, 방어막 아이템
- 3개 스테이지 구성
- 터미널 기반 게임 진행

### 기술 스택

Python, MVC 패턴, File I/O, 터미널 입출력

## 2. 지역별 교통사고 건수 분석 프로그램

공공데이터 기반 지역별 교통사고 건수를 분석하고 PyQt5 GUI로 시각화하는 데스크탑 프로그램입니다.

### 담당 역할

- CSV 데이터 로드
- 데이터 전처리
- PyQt5 기반 GUI 구현
- 데이터 조회 및 수정 인터페이스 구현

### 주요 기능

- 교통사고 CSV 데이터 파싱
- 지역별 사고 건수 조회
- GUI 기반 데이터 수정
- 사고 건수 시각화

### 기술 스택

Python, PyQt5, pandas, CSV

## 3. TCP 소켓 기반 마피아 게임

여러 클라이언트가 TCP 서버에 접속해 실시간으로 진행하는 터미널 기반 멀티플레이어 마피아 게임입니다.

### 담당 역할

- 게임 설계 참여
- 직업별 능력 구현
- 낮/밤 페이즈 전환 로직 구현
- 투표, 처형 등 이벤트 처리 로직 구현
- 전체 게임 흐름 관리 기여

### 주요 기능

- TCP 서버/클라이언트 구조
- 다중 클라이언트 접속 처리
- 마피아, 시민, 경찰 등 역할 시스템
- 낮/밤 페이즈 관리
- 터미널 기반 채팅 및 행동 처리

### 기술 스택

Python, TCP Socket, 멀티스레딩, 서버/클라이언트 구조

## 4. 로봇 경로 탐색 통합 관제 GUI

TurtleBot3를 HTTP REST API, Flask, MySQL, PyQt5 GUI와 연동한 로봇 경로 탐색 통합 관제 시스템입니다. 맵 스캐닝, 경로 탐색, 데이터 저장, GUI 모니터링을 하나의 흐름으로 구성했습니다.

### 담당 역할

- ROI 탐색 기능 협업
- 센서 감지 기능 협업
- DB 설계 및 구현
- 자체 시뮬레이션 구현

### 주요 기능

- TurtleBot3 실시간 제어
- ROI 기반 맵 스캐닝
- 장애물 감지
- Flask REST API 서버 구축
- MySQL 기반 경로 및 이벤트 데이터 저장
- PyQt5 GUI 통합 관제 화면

### 기술 스택

Python, Flask, HTTP REST API, MySQL, PyQt5, TurtleBot3, ROS, Socket

## 5. 자율주행자동차 보조안전모듈

Arduino와 ESP8266을 활용해 자율주행차 주변의 근접 거리와 환경 데이터를 감지하고, WiFi 통신으로 PC GUI에 전송하는 보조 안전 시스템입니다.

거리 기반 충돌 위험뿐 아니라 온도, 습도, 조도, 소음 등 복합 환경 데이터를 함께 감지해 위험 상황을 더 빠르게 파악하는 것을 목표로 했습니다.

### 담당 역할

- 환경 이상 감지 모듈 협업
- Arduino 센서 데이터 처리
- 보조안전모듈 보고서 정리
- 팀 통합 테스트 참여

### 시스템 구성

| 모듈 | 역할 |
| --- | --- |
| Board1 | 4방향 근접 거리 측정 및 WiFi UDP 전송 |
| Board2 | 온도, 습도, 조도, 소음 기반 환경 이상 감지 |
| Board3 | LCD, LED, 부저 기반 물리 경보 출력 |
| Python GUI | 실시간 데이터 수신 및 시각화 |
| 장애물 회피 로봇 | 초음파 센서 기반 자율 주행 및 장애물 회피 |

### 주요 기능

- 초음파 센서 기반 거리 측정
- 온습도, 조도, 소음 복합 환경 감지
- Arduino 보드 3개 연동
- UDP / HTTP 통신
- PyQt5 GUI 실시간 모니터링
- LCD, LED, 부저 경보 출력

### 기술 스택

Arduino Uno, ESP8266, HC-SR04, DHT11, LDR, Sound Sensor, Python, PyQt5, UDP, HTTP
